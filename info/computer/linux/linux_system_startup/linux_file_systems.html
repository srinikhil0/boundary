Linux Filesystem



Think of a wardrobe that has multiple shelves that can be used for storing your clothes and other accessories. These shelves help you organize the clothes by size, type, etc. The same concept applies to a filesystem, which is the embodiment of a method of storing and organizing arbitrary collections of data in a human-usable form.

Different types of filesystems supported by Linux:

  • Conventional disk filesystems: ext3, ext4, XFS, Btrfs, JFS, NTFS, vfat, exfat, etc.

  • Flash storage filesystems: ubifs, jffs2, yaffs, etc.

  • Database filesystems

  • Special purpose filesystems: procfs, sysfs, tmpfs, squashfs, debugfs, fuse, etc.
